# Learning Hard Yourself Feedback (Español)

Este repositorio es el punto central de feedback para el proyecto [Learning Hard](https://learninghard.dev/)

Usa este repositorio para:

- Reportar bugs que encuentres mientras usas o estudias el proyecto [Learning Hard](https://learninghard.dev/).
- Proponer nuevas funcionalidades, mejoras o ideas para el proyecto.
- Pedir aclaraciones o sugerir temas que creas que deberían estar mejor documentados.

## Cómo utilizar este repositorio

- Abre un **Issue de GitHub** para reportar un bug o solicitar una nueva funcionalidad.
- Aporta todo el contexto posible (pasos para reproducir el problema, comportamiento esperado, capturas de pantalla, enlaces a la página o sección afectada, etc.).

Este repositorio **no** contiene el código fuente del MOOC. Todos los comentarios y sugerencias se recopilan aquí para mantener todo organizado y fácil de seguir.
