# Learning Hard Yourself Feedback

This repository is the feedback hub for the [Learning Hard](https://learninghard.dev/) project.

Use this repository to:

- Report bugs you find while using or studying the [Learning Hard](https://learninghard.dev/) project.
- Propose new features, improvements, or ideas for the project.
- Ask for clarifications or suggest topics that you think should be better documented.

## How to use this repository

- Open a **GitHub Issue** to report a bug or request a feature.
- Provide as much context as possible (steps to reproduce, expected behavior, screenshots, links to the affected page or section, etc.).

This repository does **not** contain the source code of the MOOC itself. All feedback is collected here to keep it organized and easy to track..
